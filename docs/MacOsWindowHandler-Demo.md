# macOS 窗口按钮位置调整 - Demo 使用指南

## 在 Demo 中访问

1. 运行 TioUi.Demo 项目
2. 在左侧导航栏找到 "Platform" 分类
3. 点击 "macOS Window" 进入演示页面

## Demo 页面功能

### 1. 参数设置区域

提供三个数字输入框来调整参数：

- **X 坐标**: 按钮距离窗口左边缘的距离（推荐范围: 10 - 50）
- **Y 坐标**: 按钮的垂直偏移，负值向上（推荐范围: -20 到 20）
- **按钮间距**: 三个按钮之间的水平间距（推荐范围: 15 - 30）

### 2. 操作按钮

- **打开测试窗口**: 使用当前设置的参数打开一个新的测试窗口
- **重置默认值**: 将所有参数重置为默认值（20, -4, 20）

### 3. 预设参数

快速应用常用的参数组合：

- **标准布局** (20, -4, 20): 默认推荐的参数
- **紧凑布局** (15, -4, 15): 按钮更靠近左边缘，间距更小
- **宽松布局** (25, -4, 25): 按钮距离左边缘更远，间距更大
- **向下偏移** (20, 0, 20): 按钮不向上偏移

详细说明每个参数的作用和使用方法。

## 测试窗口

点击"打开测试窗口"后，会弹出一个新窗口：

- 窗口标题显示为 "测试窗口 - macOS 按钮位置效果"
- 窗口中央显示当前应用的参数值
- 在 macOS 系统上，你可以看到窗口左上角的三个按钮（红、黄、绿）位置已经调整
- 点击"关闭窗口"按钮关闭测试窗口

## 使用流程

1. **调整参数**: 在页面上调整 X 坐标、Y 坐标和按钮间距
2. **预览效果**: 点击"打开测试窗口"查看实际效果
3. **微调参数**: 根据测试窗口的效果，返回页面继续调整参数
4. **应用到项目**: 找到满意的参数后，参考代码示例应用到你的项目中

## 注意事项

1. **系统限制**: 此功能仅在 macOS 系统上生效，在其他系统上不会有任何效果
2. **窗口句柄**: 必须在窗口句柄可用后调用（监听 `IsVisible` 属性变化）
3. **参数范围**: 建议在推荐范围内调整参数，避免按钮超出窗口边界
4. **测试窗口**: 可以打开多个测试窗口，每个窗口使用打开时的参数设置

## 常见问题

### Q: 为什么在 Windows/Linux 上看不到效果？

A: 此功能是 macOS 特定的，仅在 macOS 系统上生效。页面会显示当前系统类型。

### Q: 如何找到最适合的参数？

A: 使用预设参数作为起点，然后根据你的窗口设计微调。建议多次打开测试窗口对比效果。

### Q: 测试窗口的按钮位置没有变化？

A: 确认你运行在 macOS 系统上，并检查控制台是否有错误信息。

### Q: 可以在运行时动态调整吗？

A: 可以，但需要重新调用 `RefreshTitleBarButtonPosition` 方法。测试窗口在打开时应用参数，不会动态更新。
